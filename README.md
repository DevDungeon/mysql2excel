mysql2excel
=============

Command-line tool to dump MySQL tables to Microsoft Excel Spreadsheets written
in Java.


## Downloading

Download the .jar

## Running

    java -jar mysql2excel-1.0.0.jar

Help information will be output when run with no command line arguments
